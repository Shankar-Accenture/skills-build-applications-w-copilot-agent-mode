OctoFit Tracker — project scaffold

This folder contains the initial scaffold for the OctoFit Tracker application.

Structure:
- backend/
  - venv/ (virtual environment — create with `python3 -m venv octofit-tracker/backend/venv`)
  - octofit_tracker/ (Django app package placeholder)
  - requirements.txt (backend Python dependencies)
- frontend/ (React/frontend placeholder)
